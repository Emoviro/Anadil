Türkçe Programlama Dili – V1 Final Tanımı
1. Dilin Amacı
Bu dil Türkçe anahtar kelimeler kullanan, okunabilir, statik tipli ve native derlenen bir programlama dilidir. V1 sürümünün amacı küçük ama sağlam bir çekirdek oluşturmak ve dili ileride büyütmeye uygun bir temel kurmaktır.
2. Temel Veri Tipleri
sayı
mantık
3. Değişken Tanımlama
Değişken tanımı şu biçimdedir:
isim: tip = değer;
Örnek:
x: sayı = 10;
y: sayı = 20;
durum: mantık = yanlış;
4. Atama
x = 30;
durum = doğru;
5. İfade Sonlandırma
Her ifade noktalı virgül ile biter.
x: sayı = 10;
yazdır(x);
x = x + 1;

8. Koşul Yapısı
eğer (x < y) {
    yazdır(x);
} değilse {
    yazdır(y);
}
9. Döngü Yapısı
Sonsuz döngü:
döngü {
    yazdır(1);
}
Koşullu döngü:
döngü (x < 10) {
    x = x + 1;
}
Sayaçlı döngü:
döngü (i: sayı = 0; i < 10; i = i + 1) {
    yazdır(i);
}
10. Döngü Kontrol İfadeleri
kır
döngü {
    eğer (x == 10) {
        kır;
    }
}
devam
döngü (i: sayı = 0; i < 10; i = i + 1) {
    eğer (i == 5) {
        devam;
    }
    yazdır(i);
}
11. Fonksiyon Tanımı
Fonksiyon tanımlamak için özel bir anahtar kelime kullanılmaz.
İsim(parametreler) {
    ...
}
İsim(parametreler) -> tip {
    ...
}
12. Fonksiyon Kuralları
Parametre varsa veri tipi yazılmak zorundadır.
Dönüş tipi belirtilirse fonksiyon değer döndürmelidir.
Dönüş tipi belirtilmezse fonksiyon dönüşsüz kabul edilir.
Topla(a: sayı, b: sayı) -> sayı {
    dön a + b;
}
YazdirDeger(x: sayı) {
    yazdır(x);
}
13. Program Giriş Noktası
Her program bir Ana() fonksiyonu içermelidir. Program çalıştırıldığında yürütme Ana() fonksiyonundan başlar.
Ana() {
    yazdır(10);
}
14. Yerleşik Fonksiyon
yazdır(değer);
15. Örnek Program
Topla(a: sayı, b: sayı) -> sayı {
    dön a + b;
}

BuyukMu(x: sayı) -> mantık {
    dön x > 10;
}

Ana() {

    x: sayı = 10;
    y: sayı = 20;

    sonuc: sayı = Topla(x, y);

    eğer (BuyukMu(sonuc)) {
        yazdır(sonuc);
    } değilse {
        yazdır(0);
    }

}
